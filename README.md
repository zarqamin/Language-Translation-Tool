# Language Translation Tool

## Overview

The Language Translation Tool is a Python-based application that translates text between different languages using the Google Gemini API. The application provides a simple graphical user interface (GUI) where users can enter text, select source and target languages, and view the translated output instantly.

## Features

* Text translation between multiple languages
* User-friendly graphical interface built with Tkinter
* Integration with Google Gemini API
* Copy translated text to clipboard
* Error handling for invalid input and API issues

## Technologies Used

* Python 3.11
* Tkinter
* Google Gemini API
* Google Generative AI Library

## Installation

1. Install Python 3.11.
2. Install the required package:

   ```
   pip install google-generativeai
   ```
3. Add your Gemini API key in the source code.
4. Run the application:

   ```
   python translator_app.py
   ```

## Working

1. Enter text in the input box.
2. Select the source language.
3. Select the target language.
4. Click the Translate button.
5. The translated text will be displayed in the output area.
6. Use the Copy button to copy the translated text.
